30 Description
31  Launder-Gibson Reynolds stress turbulence model for compressible flows.
32 
33  The default model coefficients correspond to the following:
34  \verbatim
35  LaunderGibsonRSTMCoeffs
36  {
37  Cmu 0.09;
38  kappa 0.41;
39  Clg1 1.8;
40  Clg2 0.6;
41  C1 1.44;
42  C2 1.92;
43  C1Ref 0.5;
44  C2Ref 0.3;
45  Cs 0.25;
46  Ceps 0.15;
47  sigmah 1.0; // only for compressible
48  sigmaEps 1.3;
49  sigmaR 0.81967;
50  couplingFactor 0.0;
51  }
52  \endverbatim
