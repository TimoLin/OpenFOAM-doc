30 Description
31  Launder, Reece and Rodi Reynolds-stress turbulence model for
32  compressible flows.
33 
34  The default model coefficients correspond to the following:
35  \verbatim
36  LRRCoeffs
37  {
38  Cmu 0.09;
39  Clrr1 1.8;
40  Clrr2 0.6;
41  C1 1.44;
42  C2 1.92;
43  Cs 0.25;
44  Ceps 0.15;
45  Prt 1.0; // only for compressible
46  sigmaEps 1.3;
47  sigmaR 0.81967; // only for compressible
48  couplingFactor 0.0; // only for incompressible
49  }
50  \endverbatim
