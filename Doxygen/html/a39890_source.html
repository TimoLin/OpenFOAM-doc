30 Description
31  Spalart-Allmaras one-eqn mixing-length model for compressible
32  external flows.
33 
34  Reference:
35  \verbatim
36  "A One-Equation Turbulence Model for Aerodynamic Flows"
37  P.R. Spalart,
38  S.R. Allmaras,
39  La Recherche Aerospatiale, No. 1, 1994, pp. 5-21.
40  \endverbatim
41 
42  Extended according to:
43  \verbatim
44  "An Unstructured Grid Generation and Adaptive Solution Technique
45  for High Reynolds Number Compressible Flows"
46  G.A. Ashford,
47  Ph.D. thesis, University of Michigan, 1996.
48  \endverbatim
49  by using the optional flag \c ashfordCorrection
50 
51  The default model coefficients correspond to the following:
52  \verbatim
53  SpalartAllmarasCoeffs
54  {
55  Cb1 0.1355;
56  Cb2 0.622;
57  Cw2 0.3;
58  Cw3 2.0;
59  Cv1 7.1;
60  Cv2 5.0;
61  sigmaNut 0.66666;
62  Prt 1.0; // only for compressible
63  kappa 0.41;
64  }
65  \endverbatim
