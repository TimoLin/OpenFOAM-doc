Detailed Description

Finite volume options abtract base class. Provides a base set of controls, e.g.

type scalarExplicitSource // source type active on; // on/off switch timeStart 0.0; // start time duration 1000.0; // duration selectionMode cellSet; // cellSet // points //cellZone mapRegion Note: On evaluation, source/sink options are to be added to the equation rhs
