Detailed Description

This boundary condition provides a turbulent viscosity condition for use with low Reynolds number models. It sets nut to zero, and provides an access function to calculate y+.


Patch usage

Example of the boundary condition specification: 

myPatch
{
    type            mutLowReWallFunction;
}
