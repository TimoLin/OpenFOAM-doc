30 Description
31  Launder-Gibson Reynolds stress turbulence model for incompressible flows.
32 
33  The default model coefficients correspond to the following:
34  \verbatim
35  LaunderGibsonRSTMCoeffs
36  {
37  Cmu 0.09;
38  kappa 0.41;
39  Clg1 1.8;
40  Clg2 0.6;
41  C1 1.44;
42  C2 1.92;
43  C1Ref 0.5;
44  C2Ref 0.3;
45  Cs 0.25;
46  Ceps 0.15;
47  sigmaEps 1.3;
48  sigmaR 0.81967;
49  couplingFactor 0.0;
50  }
51  \endverbatim
