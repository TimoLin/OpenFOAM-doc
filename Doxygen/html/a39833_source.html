34  Foam::incompressible::LESModel
35 
36 Description
37  Base class for all incompressible flow LES SGS models.
38 
39  This class defines the basic interface for an incompressible flow SGS
40  model, and encapsulates data of value to all possible models.
41  In particular this includes references to all the dependent fields
42  (U, phi), the physical viscosity nu, and the LESProperties
43  dictionary, which contains the model selection and model coefficients.
