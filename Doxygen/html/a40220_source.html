30 Description
31  Realizable k-epsilon turbulence model for incompressible flows.
32 
33  Model described in the paper:
34  \verbatim
35  "A New k-epsilon Eddy Viscosity Model for High Reynolds Number
36  Turbulent Flows"
37 
38  Tsan-Hsing Shih, William W. Liou, Aamir Shabbir, Zhigang Tang and
39  Jiang Zhu
40 
41  Computers and Fluids Vol. 24, No. 3, pp. 227-238, 1995
42  \endverbatim
43 
44  The default model coefficients correspond to the following:
45  \verbatim
46  realizableKECoeffs
47  {
48  Cmu 0.09;
49  A0 4.0;
50  C2 1.9;
51  sigmak 1.0;
52  sigmaEps 1.2;
53  }
54  \endverbatim
