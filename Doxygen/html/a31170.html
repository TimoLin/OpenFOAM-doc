Detailed Description

template<class BasicTurbulenceModel>
class Foam::LESModels::NicenoKEqn< BasicTurbulenceModel >

One-equation SGS model for the continuous phase in a two-phase system including bubble-generated turbulence.

Reference: 

    "One-equation sub-grid scale (SGS) modelling for Euler-Euler
    large eddy simulation (EELES) of dispersed bubbly flow"
    B. Niceno,
    M.T. Dhotre,
    N.G. Dee
    Chemical Engineering Science 63 (2008) pp. 3923-3931.

The default model coefficients correspond to the following: 

    NicenoKEqnCoeffs
    {
        Ck              0.094;
        Ce              1.048;
        alphaInversion  0.3;
        Cp              Ck;
        Cmub            0.6;
    }
